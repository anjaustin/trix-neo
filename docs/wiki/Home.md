## What is TriX?

TriX is a **deterministic AI runtime** that transforms neural networks into cryptographic "soft chips" - pattern-matching circuits that recognize inputs via Hamming distance to stored signatures.

### Key Properties

- **Deterministic**: Same input → Same output, always
- **Constant-time**: No data-dependent branches
- **Tunable latency**: Adjustable Hamming thresholds
- **Certified-ready**: Designed for FDA 510(k) and ISO 26262
