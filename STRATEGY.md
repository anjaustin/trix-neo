## The Throughline (What Makes TriX Unique)

### Core Insight
> "Mathematical truth doesn't need to be learned. Only routing needs to be learned."

**Traditional NN:** Learn everything (including XOR = a + b - 2ab)  
**TriX:** Freeze computation (XOR is frozen), learn routing (which shape fires)

### The Paradigm
```
Frozen Shapes (0 parameters) + Learned Routing (1000s of parameters) = 
Deterministic, Verifiable, Ultra-Efficient AI
```

### Why This Matters
1. **Deterministic** → Certifiable (FDA, ISO, DO)
2. **Verifiable** → Trustworthy (no black box)
3. **Efficient** → Deployable (1KB code, 64-byte state)
4. **Frozen** → Auditable (regulators can review)

**No other framework can make these claims.**
